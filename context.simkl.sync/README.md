# SIMKL Manual Sync Context Menu

Adds "Sync to SIMKL now" option to the context menu when right-clicking movies or TV episodes in your Kodi library.

## Features

- Right-click any movie or episode → "Sync to SIMKL now"
- Immediately syncs the selected item to SIMKL
- No need to wait for playback or manual addon launch
- Shows progress notification
- Reports success/failure

## Requirements

- Kodi 21.0 (Omega) or higher
- script.simkl v6.9.0 or higher (must be installed separately)
- Active SIMKL account and authentication

## Installation

1. Install script.simkl v6.9.0+ first
2. Install this context menu addon
3. Restart Kodi
4. Right-click any movie/episode → "Sync to SIMKL now"
